What it does

Built around the actual dissertation workflow

📘

Chapter Workspace

Five chapter cards track progress, notes, word counts, and goals.

📚

Research Library

Stores citation details, article notes, key findings, quotes, and how each source will be used.

✅

Task Manager

Organizes writing, reading, coursework, editing, meetings, and dissertation action items.

📅

Master Schedule

Combines dissertation deadlines, coursework, meetings, and personal commitments.

🎯

Daily Dashboard

Shows today’s priorities, currently reading, coming up items, and progress at a glance.

☁️

Cloud Saving

The working app saves privately with Firebase login, so data stays synced across devices.

How it helps

Less searching. More writing.

The dashboard is designed to reduce overwhelm by showing only what matters next. Research, tasks, chapters, and calendar items each have a clear place.
